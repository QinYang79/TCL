# TCL
Trustworthy Visual-Textual Retrieval (TIP 2025 Pytorch Code)

The datasets follow [SGRAF](https://github.com/Paranioar/SGRAF).\
The Glove models follow [NAAF](https://github.com/CrossmodalGroup/NAAF).

# Train
> sh train_VSE.sh\
> sh train_SGRAF.sh
